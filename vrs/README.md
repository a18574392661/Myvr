# VR云视家

#### 介绍
pano, VR 3D看图网站

#### 软件架构
软件架构说明
1. 后台可管理权限, vr图可根据不同账号,生成不同的套餐,在app中观看
2. 以电脑网页版本为主,查看页面的详情,列表,不同风格的分类
3. 可查看详情,在网页中查看不同的VR效果图.
4. 有一个分享模式,购买会员,二级分销奖励等模式.
